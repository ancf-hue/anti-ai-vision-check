📱 О проекте
Anti AI Vision Check — это веб-приложение для быстрой самопроверки зрения, работающее полностью офлайн. Приложение включает 13 профессиональных офтальмологических тестов в случайном порядке с мгновенной обратной связью.

✨ Возможности
🎯 13 профессиональных тестов для проверки зрения

🎲 Случайный порядок — каждый проход уникален

🔊 Звуковое сопровождение (верно/не верно) с поддержкой двух языков

🌍 Поддержка двух языков: русский и английский

📊 Система подсчета баллов и подробный результат

📤 Экспорт и сохранение результатов

🎬 Красивая анимация загрузки (фото → видео)

📱 Адаптивный дизайн для мобильных устройств (9:16)

💾 Полная офлайн-работа без интернета

📦 Установка и запуск
Требования
Любой современный браузер (Chrome, Firefox, Safari, Edge)

Для звука — поддержка HTML5 Audio
📋 Список тестов
№	Тест	Описание
1	Буквы	Проверка остроты зрения (буквы разного размера)
2	Астигматизм	Лучистая фигура для выявления астигматизма
3	Дуохром	Красный/зеленый фон для проверки рефракции
4	AI-детектор	Найди слово с цифрой вместо буквы (20+ вариантов)
5	Рабкин	Цветные кружки для проверки цветоощущения
6	Ландольт	Кольца с разрывом (определи направление)
7	Ишихара	Цветные пятна с цифрами
8	Периметрия	Проверка периферического зрения
9	Амслер	Сетка для выявления искажений сетчатки
10	Звезда Сименса	Линии, сходящиеся к центру
11	Фигуры	Найди отличающуюся фигуру (размер/цвет/форма)
12	Pulse-тест	Реакция на мигающие квадраты
13	Контраст	Определение уровня контрастности
