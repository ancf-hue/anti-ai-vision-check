👁️ Анти-ИИ Проверка зрения

Бесплатное приложение для самопроверки зрения по 13 профессиональным тестам.
📏 ДистанцияДержите телефон на расстоянии ~40 см (вытянутая рука).
👁️ ПроверкаЗакройте один глаз непрозрачной заслонкой (не ладонью). Не щурьтесь.
⚡ СкоростьМгновенный переход между тестами. Отвечайте без задержек.
🎲 Случайность13 упражнений в случайном порядке. Каждый проход уникален!
📊 РезультатПосле всех тестов вы получите общий балл и рекомендации.
